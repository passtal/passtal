저는 중앙대학교 사회학과를 졸업했습니다. 학술 발표와 토론 수업으로 리더쉽과 협업 능력을 키웠고, 사회심리와 사회조사방법론 수강을 통해 문제 해결에 있어 다각도로 접근하고 사고하는 방식을 배웠습니다. 이는 개발자로서의 역량을 키우는 것에도 큰 도움이 되었습니다.

웹 개발 분야에서 Java, JavaScript 언어를 사용하여 프로젝트를 수행하였으며,
해당 프로젝트에서는 jQuery, Ajax, MySQL 등 다양한 툴을 사용했습니다.

여타 개발자들이 놓치기 쉬운 프로젝트의 목적과 그 목적성에 맞게 탐구하는 방법을 알고 있는 것이 저의 장점입니다.
부드러운 성격으로 어느 조직에서나 뛰어난 적응력을 보이며, 팀원들과의 소통과 협업을 통해 조직 내외에서 긍정적인 영향을 주고 싶습니다.
