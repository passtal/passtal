# 👋 안녕하세요! 하루하루 발전하는 풀스택 개발자 최영우입니다.

클린 코드의 원칙을 중요하게 생각하며, 백엔드 아키텍처 설계와 더불어 사용자 친화적인 풀스택 개발 경험을 넓혀가고 있습니다.

또한, 저는 각종 철학 및 인문학적 지식을 통해 유저 친화적인 사고로 문제를 해결할 수 있는 사회학 전공 출신 신입 개발자입니다.

여타 개발자들이 놓치기 쉬운 프로젝트의 목적과 그 목적성에 맞게 탐구하는 방법을 알고 있는 것이 저의 장점입니다.

부드러운 성격으로 어느 조직에서나 뛰어난 적응력을 보이며, 팀원들과의 소통과 협업을 통해 조직 내외에서 긍정적인 영향을 주고 싶습니다.

<strong>[핵심역량]</strong>
<br>
💿 데이터 분석 도구와 사용법을 숙지하여 핵심 인사이트에 접근할 수 있습니다.

💿 Open AI Api와 같은 외부 Api를 MCP를 통해 적용한 바가 있습니다.
